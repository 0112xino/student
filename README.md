# 基于用户行为与优惠券策略的营销效果优化研究
## 一、研究背景与目的
### 1.研究背景

当前生活服务与电商行业中，优惠券已成为平台获客促活的重要手段，但因投放泛化、用户优惠感知钝化，平台面临核销率走低、ROI 不达标、用户体验下降等问题。传统广撒网式投放效率低、资源浪费，且核销率低、边际效益递减，叠加薅羊毛等恶意行为，进一步加剧成本收益失衡。

### 2.研究目标
**如何通过数据驱动的方法实现精准发券，提升核销效率，并在风险可控的前提下最大化商业价值。**

步骤：

**01  基于海量用户数据，刻画用户行为特征**

构建用户行为分析体系，识别出消费能力高但补贴依赖度低的用户，针对性设计召回策略，以提升平台的用户体验和安全性。

同时为识别风险用户，基于补贴依赖度定义异常行为，结合StandardScaler和K-Means聚类方法，识别多类异常用户群体。

**02  基于用户分群，设计差异化发券策略**

提出"分层发券+动态补贴衰减"策略，有效提升核销效率并降低资源浪费，提供一套可落地的优惠券投放优化策略，助力平台在精细化运营与风险管控间找到可持续性的平衡路径。
